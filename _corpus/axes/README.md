# Dossier des axes scientifiques

Ce dossier contient les fiches de documentation scientifique structurées par domaine de recherche.

## Convention de nommage

Chaque fiche d'axe doit respecter la convention : `AXE_<LETTRE>_<SLUG_EN_MAJUSCULES>.md`

**Exemples :**
- `AXE_A_CONSCIENCE_EM.md`
- `AXE_B_THERAPEUTIQUES_EM.md`
- `AXE_E_SCIENCECITOYENNE_EM.md`

## Structure de fiche

Tous les nouveaux axes doivent utiliser le template [`../_templates/fiche_axe.md`](../_templates/fiche_axe.md) comme point de départ.
