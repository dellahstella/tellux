# Dossier des fiches de tensions détaillées

Ce dossier contient les fiches de tension individuelle quand une tension devient suffisamment complexe pour justifier sa propre documentation.

## Quand créer une fiche de tension ?

Une tension justifie un fichier dédié dans ce dossier si elle :
- Implique **3+ axes scientifiques** ou plus
- Nécessite une **analyse comparée détaillée** des positions
- A une **histoire de débat** remontant plusieurs années
- Affecte directement le **design de modèle Tellux**

Pour les tensions simples ou récentes, rester dans [`../TENSIONS.md`](../TENSIONS.md) à la racine.

## Convention de nommage

Chaque fiche de tension doit respecter : `TENSION_<SLUG>.md`

**Exemples :**
- `TENSION_ELF_LEUCEMIES.md`
- `TENSION_ICNIRP_NON_THERMIQUE.md`
- `TENSION_EHS_CAUSALITE.md`

## Structure de fiche

Utiliser le template [`../_templates/tension.md`](../_templates/tension.md) comme point de départ.

## Tensions actuelles

Tensions pré-identifiées (T1-T8) en `../TENSIONS.md`. Aucune fiche détaillée créée pour le moment ; fiches seront créées en ce dossier au fur et à mesure des révisions du corpus.
