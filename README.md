# 🌙⚡ Tellux – Cartographie CEM & Patrimoine Corse

**Plateforme open-source interactive pour sensibilisation aux champs électromagnétiques et valorisation du patrimoine mégalithique corse.**

## 🎯 Vue d'ensemble

**Tellux** permet aux habitants, mairies et agriculteurs de Corse de visualiser en temps réel :

- **Exposition CEM naturels et anthropiques** (antennes ANFR, lignes électriques, magnétisme terrestre)
- **196 églises romanes** avec orientations astronomiques (Moracchini-Mazel 1967-1992)
- **66+ sites mégalithiques** (dolmens, menhirs, alignements) avec analyses géophysiques
- **Anomalies géologiques** (magnétisme IGRF, anomalies crustales EMAG2v3)
- **Mesures crowdsourcées** (citoyens contributeurs avec Trifield TF2, apps)

**Objectif** : Science citoyenne transparente, sensibilisation santé environnementale, valorisation patrimoniale Corse.

---

## ✨ Fonctionnalités

### Carte interactive Leaflet
- Zoom jusqu'au niveau rue/maison
- Couches superposables (antennes, lignes électriques, patrimoine, magnétisme)
- Popups détaillées coordonnées GPS, descriptions, sources

### Scoring géobiologique
- Échelle 0-10 (naturel vs perturbations humaines)
- Analyse combinée : proximité antennes, lignes, anomalies naturelles
- Transparent : voir le calcul exact

### Crowdsourcing citoyen
- Saisir mesures terrain (ELF, RF, magnétique)
- Géolocalisation auto
- Points agrégés temps réel

### Corpus scientifique
- **70+ études peer-reviewed** (2013-2026)
- Agriculture régénérative, EM, cycles lunaires, archéoastronomie
- Voir dossier `corpus/` complet
