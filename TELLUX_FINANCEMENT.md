## Budget prévisionnel type

Ce budget est un squelette réutilisable pour plusieurs guichets. Les montants sont indicatifs et doivent être ajustés selon le guichet visé.

| Poste | Description | Montant estimé (12 mois) |
|---|---|---|
| Temps porteur | Développement, recherche, coordination (valorisé) | À chiffrer |
| Hébergement | Cloudflare Pages (gratuit), Supabase (plan gratuit → pro si besoin) | 0 – 300 €/an |
| Domaine | Nom de domaine personnalisé (optionnel) | 15 €/an |
| Matériel de mesure | Capteurs EM terrain (si campagne de mesure prévue) | 500 – 2 000 € |
| Déplacements | Sites mégalithiques, réunions partenaires (Corse) | 500 – 1 500 € |
| Communication | Landing page (Framer), supports de présentation | 0 – 500 € |
| Outils numériques | Claude (abonnement), N8N (si self-hosted VPS), GitHub (gratuit) | 200 – 600 €/an |
| Comptabilité | Si nouvelle structure ou obligations renforcées | 0 – 3 000 €/an |
| **Total indicatif** | | **1 200 – 8 000 €/an** |

*Note : le poste « temps porteur » est souvent le plus important en montant valorisé. Les financeurs publics acceptent généralement la valorisation du temps bénévole ou du temps de travail du porteur.*
