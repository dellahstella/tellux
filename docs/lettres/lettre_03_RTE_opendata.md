# Lettre 03 — RTE Open Data

**Objet** : Demande d'orientation sur la stabilité et la documentation des flux eco2mix et jeux de données RTE utilisables pour le projet Tellux Corse

**Destinataire** : Réseau de Transport d'Électricité (RTE), direction Open Data

**Expéditeur** : Soleil, porteur du projet Tellux Corse, SARL Stella Canis Majoris, Bastia

**Date** : 20 avril 2026

---

Madame, Monsieur,

Je me permets de vous solliciter au sujet de vos plateformes de données ouvertes, dans le cadre du projet Tellux Corse que je développe depuis Bastia.

Tellux est une plateforme cartographique publique consacrée à l'électromagnétisme environnemental en Corse. Elle agrège des données publiques issues de plusieurs institutions de référence. Le champ magnétique basse fréquence à 50 hertz, l'un des quatre domaines traités, dépend notamment de la charge sur le réseau électrique. La variabilité temporelle (base, pointe, saisonnalité) constitue une information importante pour contextualiser les modélisations spatiales, y compris en Corse où la spécificité insulaire (EDF SEI) se combine, pour certaines analyses, à des références continentales publiées par RTE.

Les jeux de données accessibles via eco2mix et les plateformes ouvertes de RTE sont d'un grand intérêt pour ce projet. Je souhaiterais néanmoins m'assurer, avant toute intégration automatisée, de plusieurs points :

- la stabilité dans le temps des points d'accès programmatiques (API, URL d'export, formats),
- les conditions d'usage acceptables pour une plateforme tierce publique et non commerciale, même lorsque cette plateforme se réclame d'un usage pédagogique et territorial,
- l'existence d'une documentation de référence pour les granularités temporelles et spatiales pertinentes (pas de temps, périmètre régional, périmètre territorial corse lorsqu'il est disponible),
- l'existence éventuelle d'un point de contact pour les projets de cartographie qui souhaitent citer RTE comme source et souhaitent être notifiés en cas de changement de schéma.

Tellux s'engage à citer RTE comme source explicite sur chaque couche utilisant vos données, à indiquer le millésime et la licence, et à ne pas publier de représentation qui pourrait être confondue avec une mesure directe du champ 50 Hz en un point donné. Les données RTE sont utilisées comme facteur de modulation temporelle d'une modélisation spatiale, pas comme une mesure d'exposition. Le projet ne fait aucune interprétation sanitaire individuelle.

Je serais reconnaissant de pouvoir :

- être orienté vers le bon interlocuteur au sein de RTE pour les questions de stabilité et de documentation des flux,
- recevoir, si cela est possible, une confirmation écrite qu'un usage cartographique public et non commercial comme celui de Tellux est compatible avec les conditions générales de vos plateformes de données ouvertes,
- être tenu informé des évolutions programmées qui affecteraient les jeux de données aujourd'hui disponibles.

Je me tiens à votre disposition pour tout échange et pour vous présenter plus précisément l'architecture du projet, les sources déjà intégrées et les points d'incertitude actuels.

Je vous prie d'agréer, Madame, Monsieur, l'expression de mes salutations respectueuses.
