# Tellux — Analyses statistiques

Dossier des analyses reproductibles menées sur le corpus Tellux.

## Conventions

- Un rapport markdown par analyse, avec date et version
- Un script Python reproductible (seed fixé)
- Graphiques PNG liés au rapport
- Indication des dépendances et de la commande d'exécution dans le rapport lui-même

## Analyses versionnées

- `TELLUX_CORRELATION_MEGALITHS_v1.md` (2026-04-17) — Corrélation mégalithes × failles × piézoélectricité. Hypothèse H1 du corpus. Résultat préliminaire positif (p < 1e-4 sur indicateurs non-circulaires). Voir rapport.
